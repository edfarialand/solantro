## 🛠️ Development

### Prerequisites

- Solana CLI
- Anchor Framework
- Rust

### Build and Deploy

```bash
# Install dependencies
yarn install

# Build the program
anchor build

# Deploy to devnet
anchor deploy --provider.cluster devnet

# Test
anchor test
```

### Create Token with Metadata

```bash
# Run the token creation script
./create_token_with_metadata.sh
```
